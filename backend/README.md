## 🎯 За проекта

HelpChain.bg е иновативна уеб платформа, разработена с Flask, която цели да създаде мост между хората, нуждаещи се от помощ, и доброволците, готови да предложат подкрепа. Платформата подкрепя многоезичност и е достъпна за всички потребители.

### 🚀 Ключови характеристики

- **🔐 Сигурност на първо място**: Пълна криптография на данни и защитени паароли
- **📊 Analytics Dashboard**: Разширена система за анализ и мониторинг
- **🌍 Многоезичност**: Подкрепа за български и английски език
- **📧 Email система**: Автоматизирани известия и комуникация
- **🔧 Admin панел**: Пълноценен административен интерфейс
- **📱 Responsive Design**: Оптимизирана за всички устройства

## 🚀 Бърз старт

```bash
# Клониране на repository
git clone https://github.com/stellababy2004/HelpChain.bg.git
cd HelpChain.bg/backend

# Създаване на virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# Инсталиране на dependencies
pip install -r requirements.txt

# Конфигуриране на environment variables
cp .env.example .env
# Редактирайте .env файла с вашите настройки

# Инициализиране на базата данни
python init_db.py

# Стартиране на приложението
python run.py
```
