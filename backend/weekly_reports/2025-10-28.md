## Deployment snippets

- **Nginx static cache**: Готов nginx snippet за статични assets с cache headers — `deploy/nginx_static_cache.conf`. Включи го в server block:
  ```nginx
  include /path/to/your/repo/deploy/nginx_static_cache.conf;
  ```
- Дълъг TTL за fingerprinted assets, по-кратък за други статични файлове.

## Quick Start

- Стартирай бекенда:
  ```pwsh
  python app.py
  ```
- Smoke тест:
  ```pwsh
  python scripts/smoke_fts.py
  ```
- Service worker cache note: Ако landing page не се обновява след deploy, unregister SW и hard refresh:
  - DevTools → Application → Service Workers → Unregister, после Ctrl+Shift+R
  - Или в Console:
    ```js
    navigator.serviceWorker
      .getRegistrations()
      .then((rs) => rs.forEach((r) => r.unregister()));
    caches
      .keys()
      .then((keys) => Promise.all(keys.map((k) => caches.delete(k))));
    ```

## Troubleshooting

- Проверка на FTS статус:
  ```pwsh
  pwsh ./scripts/dev.ps1 -Task fts-status
  ```
- Smoke тест:
  ```pwsh
  pwsh ./scripts/dev.ps1 -Task smoke
  ```
- Reset на локалната DB:
  ```pwsh
  Remove-Item .\instance\volunteers.db -Force
  python app.py
  ```

## How-to

- Генериране на силен secret:
  ```pwsh
  python -c "import secrets; print(secrets.token_urlsafe(48))"
  ```
- Добави го в `.env`:
  ```env
  HELPCHAIN_SECRET_KEY=your_long_random_value_here
  ```
