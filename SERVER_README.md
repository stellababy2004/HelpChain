# HelpChain Server Quick Start

## 🚀 Бързо стартиране

### Вариант 1: Batch скриптове (Препоръчително за Windows)

**Стартиране на сървъра:**

```bash
# Стартиране в нов прозорец (не блокира терминала)
start_server_bg.bat

# Или стартиране в същия прозорец (блокира терминала)
start_server.bat
```

**Спиране на сървъра:**

```bash
stop_server.bat
```

### Вариант 2: Python скрипт

```bash
python start_server.py
```

### Вариант 3: Ръчно стартиране

```bash
python -c "
import sys
sys.path.insert(0, 'backend')
from appy import app
app.run(host='127.0.0.1', port=5000, debug=False)
"
```

## 🌐 Достъп до приложението

След стартиране, отвори браузъра на:

- **Главна страница:** http://127.0.0.1:5000
- **Админ панел:** http://127.0.0.1:5000/admin_login
- **Вход за доброволци:** http://127.0.0.1:5000/volunteer_login
- **API чатбот:** http://127.0.0.1:5000/api/chatbot/message

## 🛠️ Отстраняване на проблеми

Ако сървърът не стартира:

1. **Провери дали порт 5000 е свободен:**

   ```bash
   netstat -ano | findstr :5000
   ```

2. **Спри всички Python процеси:**

   ```bash
   taskkill /f /im python.exe
   ```

3. **Стартирай отново:**
   ```bash
   start_server_bg.bat
   ```

## 💡 Съвети

- Използвай `start_server_bg.bat` за нормална работа
- Използвай `start_server.bat` само ако искаш да виждаш лог файловете в реално време
- Сървърът автоматично се рестартира при грешки в кода
- Всички настройки са в `backend/appy.py`
