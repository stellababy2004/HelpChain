{{ _("Questions fréquentes") }}

{{ _("Cette page répond aux principales questions concernant le fonctionnement de la plateforme HelpChain.") }}

{{ _("Qu’est-ce que HelpChain ?") }}

{{ _("HelpChain est une infrastructure numérique de coordination conçue pour les structures, collectivités, associations et professionnels impliqués dans le traitement organisé des situations sociales.") }}

{{ _("Elle permet de structurer l’orientation, le suivi et le pilotage des situations dans un cadre opérationnel partagé.") }}

{{ _("Qui peut utiliser la plateforme ?") }}

{{ _("HelpChain s’adresse en priorité aux structures, associations, collectivités et professionnels habilités qui doivent coordonner des situations dans un cadre structuré.") }}

  • {{ _("collectivités et services publics locaux") }}
  • {{ _("associations et structures sociales") }}
  • {{ _("professionnels habilités") }}
  • {{ _("structures locales impliquées dans la coordination") }}

{{ _("Selon l’organisation mise en place, des citoyens ou des partenaires externes peuvent aussi faire partie du circuit de signalement ou d’entrée de la demande.") }}

{{ _("La finalité principale reste la coordination structurée, le suivi et la visibilité opérationnelle pour les organisations qui portent le traitement.") }}

{{ _("La plateforme remplace-t-elle les structures existantes ?") }}

{{ _("Non.") }}

{{ _("HelpChain ne remplace pas les structures existantes. La plateforme agit comme un cadre complémentaire pour structurer la coordination, clarifier les responsabilités et améliorer le suivi entre acteurs autorisés.") }}

{{ _("Les données personnelles sont-elles protégées ?") }}

{{ _("Oui.") }}

{{ _("HelpChain est conçu dans le respect des principes du Règlement Général sur la Protection des Données (RGPD).") }}

{{ _("Les données collectées sont limitées aux informations nécessaires au fonctionnement du service.") }}

{{ _("Faut-il installer un logiciel ?") }}

{{ _("Non.") }}

{{ _("HelpChain est accessible directement via un navigateur internet et ne nécessite aucune installation spécifique.") }}

{{ _("Qui gère la plateforme ?") }}

{{ _("La plateforme est développée et maintenue dans le cadre du projet HelpChain, avec pour objectif de soutenir la coordination opérationnelle entre structures, collectivités, associations et professionnels habilités.") }}
