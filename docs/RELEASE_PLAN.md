# Release Plan – Weekly Cycle

## Rhythm

- **Monday:** Review issues, assign milestones.
- **Wednesday:** Development, testing.
- **Friday:** Merge PRs & deploy.

## Branching

- `feature/` → new features
- `fix/` → bug fixes
- `chore/` → maintenance

## Automation

- CI/CD: Lint → Test → Build → Deploy.
- Dependabot auto-updates weekly.
- Monitoring: Site24x7 APM + RUM active.

## Retrospective

Every Friday → document week summary in Notes.
