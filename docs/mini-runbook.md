# HelpChain Mini Runbook

1. Run:
   `pytest tests/test_request_flow_smoke.py`
2. Run:
   `pytest tests/test_system_health_smoke.py`
3. Open:
   `/health`
4. Open:
   `/admin/sanity`
5. Create one request from the public flow.
6. Verify the request in admin:
   - `/admin/requests` (list)
   - `/admin/requests/<id>` (detail)
